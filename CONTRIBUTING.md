# Contributing to Irie Trade Bot

🎉 Thank you for your interest in contributing! This project is actively maintained and open to improvements.

## How Can I Contribute?

- **Report bugs** – Open an issue and describe the bug, including logs and steps to reproduce.
- **Suggest features** – Open an issue with the “enhancement” label.
- **Submit pull requests** – Fork the repo, create a feature branch, and submit a PR. Please:
  - Follow the existing code style (PEP8, type hints, `loguru` for logging).
  - Add tests for new functionality (if applicable).
  - Update the README or documentation if you change behaviour.
- **Improve documentation** – Typos, unclear instructions, or missing sections are all valid contributions.

## Getting Started

1. Fork the repository.
2. Clone your fork: `git clone https://github.com/<your-username>/IrieTrade.git`
3. Create a branch: `git checkout -b feature/my-feature`
4. Make your changes and commit: `git commit -m "feat: add my feature"`
5. Push to your fork and open a pull request.

## Community Guidelines

- Be respectful and constructive.
- Keep discussions focused on the project.
- If you’re unsure about something, ask before coding.

## Reporting a Bug

Use the [Bug Report template](https://github.com/Native-254/IrieTrade/issues/new?template=bug_report.md) (coming soon) or just include:

- Your Python version and OS
- The exact error message / log output
- Steps to reproduce

## Feature Requests

Use the [Feature Request template](https://github.com/Native-254/IrieTrade/issues/new?template=feature_request.md) (coming soon) or describe:

- The problem you want to solve
- How your feature would work
- Any alternatives you’ve considered

Thank you for helping make Irie better!
